# Social Setup Payloads

These packets turn the expanded distribution strategy into copy/paste-ready
setup instructions for each social surface.

They are intentionally practical:

- exact public name or handle,
- category or page type,
- default link,
- profile image and banner asset,
- bio/about copy,
- starter pinned post,
- Smarsh handoff,
- Metricool Brand target,
- what still requires David's logged-in click.

Do not publish starter posts until the surface exists, David has connected it to
Smarsh, and the matching Metricool Brand/platform connection is verified.

## Launch Order

1. Create or verify the destination surface.
2. Add the profile image, banner, bio/about copy, and default link.
3. Add David as owner/admin and Stephenee as backup admin where supported.
4. David connects the surface to Smarsh.
5. Connect the surface to the matching Metricool Brand.
6. Only then allow the content engine to schedule to that property.
