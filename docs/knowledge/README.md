## Natural Shorthand

When David says "make it sound like me," that means more than matching tone. Use the full David voice and point-of-view lens:

- Read the [Voice Guide](./voice-guide.md) for sentence-level posture.
- Use the [Interpretation Layer](./interpretation-layer.md) before drafting.
- Filter the work through the [Point Of View Map](./point-of-view-map.md).
- For hooks, posts, emails, or video ideas, apply the [Social Content Principles](./social-content-principles.md).

In practice, "sound like me" means preserve David's judgment, specificity, and live tension. Do not sand the work into generic financial-advisor language.
