# Launch Cleanup Plan

## Pass 1: Make The Workspace Trustworthy
- Use one preview port and one preview command.
- Stop relying on stale built-file patches.
- Fix local build and check commands.
- Confirm source, build output, and browser preview match.

## Pass 2: Technical SEO And Launch Crawl
- Crawl all public routes.
- Confirm status codes, redirects, canonical tags, and noindex rules.
- Clean sitemap and robots.
- Verify /brands/talley-wealth/* duplicate behavior.
- Confirm old URLs redirect properly.
- Check title and description coverage.
- Check schema output.
- Verify forms, guide download, contact paths, and client portal links.

## Pass 3: Performance And Content Polish
- Optimize oversized images, especially hero and team assets.
- Review homepage and core conversion path.
- Review services, who-we-help, pricing, get-started, and contact.
- Remove weak/stale pages from sitemap or noindex them.
- Tighten outdated or bloated copy.
- Produce final launch-readiness summary.
